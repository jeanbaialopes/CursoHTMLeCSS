# Português

## Sobre o Projeto
Este repositório compila os exercícios e projetos desenvolvidos durante o curso de HTML 5 e CSS 3 oferecido pelo [Curso em Vídeo](https://www.cursoemvideo.com). O objetivo é documentar o progresso e consolidar o aprendizado em tecnologias fundamentais para o desenvolvimento web, como HTML5 e CSS3, através de exemplos práticos e projetos guiados.

## Tecnologias Utilizadas
As seguintes tecnologias foram utilizadas no desenvolvimento e estudo deste projeto:

-   **HTML5** ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=flat&logo=html5&logoColor=white)
-   **CSS3** ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=flat&logo=css3&logoColor=white)

## Pré-requisitos
Para visualizar e interagir com os arquivos deste repositório, você precisará de:

*   Uma IDE de sua preferência (ex: Visual Studio Code) ou um editor de texto simples.
*   Um navegador web moderno (Chrome, Firefox, Edge, Safari, etc.).

## Instalação e Execução
Este projeto consiste em arquivos estáticos de HTML e CSS, portanto, não requer um processo de instalação complexo ou comandos de execução específicos.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/jeanbaialopes/curso-html-css.git
    
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd curso-html-css
    ```
3.  **Abra os arquivos HTML:**
    Basta abrir qualquer arquivo `.html` diretamente no seu navegador web.

## Exemplo de Uso
Os arquivos estão organizados por módulos e projetos. Para ver um exemplo de como os conceitos são aplicados, navegue pelas pastas e abra os arquivos HTML correspondentes. Cada arquivo representa uma etapa ou um projeto específico do curso.

## About the Project
This repository compiles the exercises and projects developed during the HTML 5 and CSS 3 course offered by Curso em Vídeo (https://www.cursoemvideo.com). The objective is to document progress and consolidate learning in fundamental web development technologies, such as HTML5 and CSS3, through practical examples and guided projects.

## Technologies Used
The following technologies were used in the development and study of this project:

-   **HTML5** ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=flat&logo=html5&logoColor=white)
-   **CSS3** ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=flat&logo=css3&logoColor=white)

## Prerequisites
To view and interact with the files in this repository, you will need:

*   An IDE of your choice (e.g., Visual Studio Code) or a simple text editor.
*   A modern web browser (Chrome, Firefox, Edge, Safari, etc.).

## Installation and Execution
This project consists of static HTML and CSS files, therefore, it does not require a complex installation process or specific execution commands.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/jeanbaialopes/curso-html-css.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd curso-html-css
    ```
3.  **Open the HTML files:**
    Simply open any `.html` file directly in your web browser.

## Usage Example
