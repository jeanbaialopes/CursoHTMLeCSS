Conceitos Básicos

Altura x

A altura x é a altura base para qualquer fonte, ela determina o tamanho vertical da fonte para todas as letras minúsculas.

Altura Ascendente

É a altura das letras que extravazam a altura-x para cima, como as letras: i, b e d.

Altura Corpo

É a altura total entre a altura Ascendente e a Descendente.

Haste

Retângulo vertical de uma letra.

Arco

É um arco, não necessáriamente simétrico, que une duas hastes.

Vértice

Um ângulo agudo. Presente no topo do A e na base do V.

Braço

Elemento que sai de uma Haste e se eleva.

Pé

Depois da perna, geralmente onde aparecem a serifa.

Olho

Circunferência fechada de uma letra. Como em Q, q, g, e e p.

Cauda

Parte decendente do glifo.

Altura das Maiúsculas

É a altura que as letras maiúsculas ocupam.

Altura Descendente

É a altura das letras que extravazam a altura-x para baixo, como as letras: ç, q, j e p.

Serifa

É o prolongamento de uma letra que facilita a leitura.

Filete

Retângulo horizontal de uma letra. É uma forma de unir hastes e arcos.

Esporão

Um tipo de "salto" para a letra, para esta não parecer que vai cair. Geralmente aparece na letra b.

Terminal

Um arco que não conecta duas astes, muito visto na letra r.

Perna

Elemento que sai de uma haste ou braço e se rebaixa.

Barriga

Arco longo, como na letra b.

Orelha

Prosongamento na parte superior de uma letra.

Glifo é cada letra, que compõe uma fonte - um conjunto de glifos. Tipos diferentes de fontes formam uma familia tipográfica. Tipos diferentes de familias tipográficas criam categorias tipográficas, trazendo fontes serifadas, sans-serif (sem serifa), monoespaçadas, handwriting ou display. Podemos combinar várias características de fontes.
